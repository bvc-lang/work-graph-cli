# work-graph/cli

npm: [`@work-graph/cli`](https://www.npmjs.com/package/@work-graph/cli) v0.2.5

Install Work Graph into any project with local BVC work items, Cursor MCP config and backlog UI.

Monorepo: [bvc-lang/work-graph](https://github.com/bvc-lang/work-graph)

## Quick start

```bash
npx @work-graph/cli init .
npm install
npm run workgraph:ui
```
